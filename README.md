# 📄 Resume Ranking System

## 📌 Overview

The Resume Ranking System is an AI-powered application that evaluates and ranks resumes based on their relevance to a given job description. It uses Natural Language Processing (NLP) and Machine Learning techniques to calculate similarity scores and help recruiters identify the most suitable candidates efficiently.

---

## ✨ Features

- 📄 Upload multiple resumes
- 📝 Analyze job descriptions
- 🤖 AI-based resume ranking
- 📊 Similarity score calculation
- ⚡ Fast and automated screening
- 💻 User-friendly interface

---

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/resume-ranking-system.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```

---

## ▶️ Usage

1. Launch the application.
2. Upload one or more resumes.
3. Enter or upload a job description.
4. Click **Rank Resumes**.
5. View ranked resumes with similarity scores.

---

## 📊 Output

The application ranks resumes based on their similarity to the provided job description, making the recruitment process faster and more efficient.
